DATA TYPES ADDED
────────────────────────────────────────────────────────────────────────────────

In src/types/api.ts:

Variant Interface:
  • id: string - Unique variant ID
  • name: string - Variant name (e.g., "Small", "Large", "Zinger")
  • priceAdjustment?: number - Price difference from base
  • icon?: string - Optional emoji or icon
  • description?: string - Short description
  • isDefault?: boolean - Mark as default selection

Modifier Interface:
  • id: string - Unique modifier ID
  • name: string - Modifier name (e.g., "Extra Cheese", "Bacon")
  • price: number - Cost of this modifier
  • icon?: string - Optional emoji or icon
  • category?: string - Category (e.g., "Sauces", "Toppings")
  • isDefault?: boolean - Pre-selected by default

ModifierGroup Interface:
  • id: string - Group ID
  • name: string - Group name (e.g., "Sauces", "Extras")
  • modifiers: Modifier[] - Array of modifiers in group
  • isRequired?: boolean - Must select one
  • maxSelections?: number - Maximum selections allowed (0 = unlimited)

Item Type Extended:
  • variants?: Variant[] - Size/type options
  • modifierGroups?: ModifierGroup[] - Customization options
