DATA TYPES ADDED (src/types/api.ts)
────────────────────────────────────────────────────────────────────────────────

Variant Interface:
  - id: string (unique identifier)
  - name: string (e.g., "Small", "Medium", "Large")
  - priceAdjustment?: number (additional cost)
  - icon?: string (emoji or icon like 🍔)
  - description?: string (short description)
  - isDefault?: boolean (pre-selected)

Modifier Interface:
  - id: string (unique identifier)
  - name: string (e.g., "Extra Cheese", "Bacon")
  - price: number (modifier cost)
  - icon?: string (emoji like 🧀)
  - category?: string (organizational category)
  - isDefault?: boolean (pre-selected)

ModifierGroup Interface:
  - id: string (group identifier)
  - name: string (e.g., "Sauces", "Extras")
  - modifiers: Modifier[] (modifiers in group)
  - isRequired?: boolean (must select one)
  - maxSelections?: number (selection limit)

Item Type Extended:
  - variants?: Variant[] (size/type options)
  - modifierGroups?: ModifierGroup[] (customization groups)

HOW IT WORKS - USER EXPERIENCE
────────────────────────────────────────────────────────────────────────────────

1. User opens product detail page (e.g., Classic Burger)
2. Views product image, description, and base price ($16.99)
3. Sees CustomizationPanel with variants and modifiers
4. User can:
   - Scroll through variant options (Single/Double/Triple)
   - Select a variant (visual feedback with color + dot)
   - Price adjusts (e.g., Double = +$5.99)
5. User can select modifiers:
   - Tap to add/remove extras (Bacon, Cheese, Avocado, etc.)
   - Each modifier shows individual price
   - Multiple modifiers can be selected
6. Real-time price updates:
   - CustomizationPanel shows breakdown:
     * Base Price: $16.99
     * Size Adjustment: +$5.99
     * Extras: +$4.48 (Bacon + Cheese + Avocado)
     * Total: $27.46
7. Add to Cart button reflects customized price
8. User adds item with all customizations to cart


PRICE CALCULATION LOGIC
────────────────────────────────────────────────────────────────────────────────

Base Price: product.price (e.g., $16.99)

Selected Variant Price Adjustment:
  - If variant has priceAdjustment, add it
  - Example: Double variant = +$5.99

Modifiers Total:
  - Sum of all selected modifier prices
  - Each modifier has individual price
  - Example: Bacon ($2.49) + Cheese ($1.49) + Avocado ($2.99) = $6.97

Total Price Calculation:
  totalPrice = basePrice + variantAdjustment + modifiersTotal
  Example: $16.99 + $5.99 + $6.97 = $29.95

Add to Cart Amount:
  cartPrice = totalPrice * quantity
  Example: $29.95 * 2 = $59.90

HOW TO USE IN YOUR APP
────────────────────────────────────────────────────────────────────────────────

For Products WITHOUT Variants/Modifiers:
  • CustomizationPanel won't render (conditional check)
  • ProductDetailScreen works as before
  • No changes needed to existing products

For Products WITH Variants/Modifiers:
  • Add variants array to Item object
  • Add modifierGroups array to Item object
  • CustomizationPanel automatically appears
  • Price calculation handles automatically
  • Users can customize before adding to cart

Example Product with Customization:
  {
    id: 'product-1',
    name: 'Deluxe Burger',
    price: 14.99,
    variants: [
      { id: 'v1', name: 'Single', priceAdjustment: 0 },
      { id: 'v2', name: 'Double', priceAdjustment: 5.99 }
    ],
    modifierGroups: [
      {
        id: 'g1',
        name: 'Extras',
        modifiers: [
          { id: 'm1', name: 'Cheese', price: 1.49 },
          { id: 'm2', name: 'Bacon', price: 2.49 }
        ]
      }
    ]
  }
